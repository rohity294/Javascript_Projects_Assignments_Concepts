A Scientific Calculator

Kindly note:- For Sin,Cos,Tan,Log,Exponent,square,cube,square root,cube root etc operations, type number and hit relevant button.
Eg: 45 Sin, 100 Log, 56 Exponent, 4 square, 6 cube, 16 square root, 125 cube root
Also note that using % in any evaulation expression would mean 'modulus' and provide remainder. Use separate 'percent' button if you intend to evaluate percentage.
